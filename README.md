## Usage

### Environment Dependencies

- install `maven`
- install `java 8`
- `mvn install -Dmaven.test.skip=true` download prerequisites dependencies
without test.

### Synopsis

```bash
# After git clone the respository
mvn spring-boot:run
```

Then the spring boot application is running on `localhost:8080`.

### How to test

#### command
```bash
mvn test
```

or

```bash
mvn surefire:test
```
